### Usage

1. Clone the repo:

```sh
git clone git@github.com:my-user/my-app.git
cd ./myapp
```

2. Install npm dependencies:

```sh
yarn
```

3. Install pod dependencies:

```sh
npx pod-install ios
```

4. Run:

```sh
yarn android
yarn ios
```
